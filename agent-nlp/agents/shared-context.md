## Self-Improving Loop
```
SUBMIT 10 -> ANALYZE (efficiency_analyzer.py) -> DIAGNOSE (self_improve.py) -> FIX -> DEPLOY -> SUBMIT 10
```
After each batch, the analyzer reads Cloud Run logs and generates ranked fix suggestions. The fixer implements the top fix. Repeat.

## Boris Workflow (mandatory, every code change)
```
EXPLORE: What is the current bottleneck? (read logs, check scores)

CODE:    Implement the change

SIMPLIFY: code-simplifier cleans up
VALIDATE: python3 -c "import ast; ast.parse(...)" + deploy + smoke test
COMMIT:  git commit with score delta
```
No exceptions. Every change follows this loop.
